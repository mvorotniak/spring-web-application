### Spring 5 Demo Web Application
This is a Spring 5 Demo Web Application created under scope of passing the _Udemy_ course `Spring Framework 5: Beginner to Guru` by John Thompson.

### Technologies covered
- Spring Framework 5
- Spring Boot 2
- Spring Data JPA
- Spring MVC
- Spring MockMVC
- Spring WebFlux
- Spring Web Client
- Spring Data MongoDB
- Spring RestDocs
- Hibernate
- Project Lombok
- MapStruct
- Maven
- Gradle
- JUnit 4 and JUnit 5
- Mockito
